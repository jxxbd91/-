## js中数组的遍历方式

### for
for 循环遍历是最普通的一种方式，通过数组中自带的索引进行数组元素的获取，这种方式可以支持 break、continue、return的打断形式。具体如下：
```
for (var i = 0; i < arr.length; i++) {
	// 通过索引进行数组元素的获取
	console.log(arr[i])
	if ()
}
```
